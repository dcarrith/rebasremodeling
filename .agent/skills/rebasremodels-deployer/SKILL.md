# Rebasremodels Deployer Skill

Packages and deploys the Reba's Remodeling Vite/React site to a local Kubernetes cluster context, serving the compiled static assets via Nginx.

## Prerequisites

- Docker installed and running
- `kind` cluster running (Targeting cluster named: `higgs-cluster`)
- `kubectl` installed and configured

## Workflows

### Build and Deploy

```bash
./.agent/skills/rebasremodels-deployer/scripts/deploy.sh
```

**Actions performed by this script:**
1. Builds the static Vite assets and creates a production Nginx Docker image (`rebasremodels-app:latest`).
2. Loads the newly built image into the local `kind` cluster.
3. Applies the Kubernetes Deployment and Service manifests.
4. Restarts the rollout and waits for deployment completion.

**After deployment:**
You can access the application locally by running:

```bash
kubectl port-forward service/rebasremodels-web 8080:80
```
Then visit [http://localhost:8080](http://localhost:8080) in your browser.

## Architecture

- **rebasremodels-web**: An Nginx container serving the static compiled React/Three.js assets generated by `npm run build`.
- **Deployment Strategy**: Recreates the pods to ensure the latest image tag is pulled within the local cluster.
